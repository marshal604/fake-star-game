# Dialogue Review Report — main-story + sign-suman

## 整體評估

整體 AI 感中等偏高,`sign-suman.ts` 已有一點蘇嫚君的結巴和迷糊感,但 `main-story.ts` 從育幼院事件開始常變成「劇情摘要 + 功能性台詞」。主要問題是情緒說得太直、生活細節不足、選項回應結構太對稱,中後段尤其缺 90s 台灣演藝/育幼院現場感。

## 高 AI 感重點段落(>= 4 分)

### 1. `e107_a` — 大掃除後感謝

原句太像標準正向回饋:「謝謝你陪我來。看到孩子們的笑容,我什麼都忘了。」

建議改寫:

```ts
emotion: 'happy',
text: '「欸,你剛剛被小志叫叔叔的時候,臉真的好好笑……啊,不是啦!我是說……謝謝你今天來。窗戶那邊如果只有我,大概擦到天黑吧。」'
```

### 2. `e109_a` — 聖誕夜告白前

原句直接說「幸福」,情感太明牌。

建議改寫:

```ts
emotion: 'shy',
text: '「今天的燈泡有幾顆一直閃,可是……不知道為什麼,我覺得比百貨公司的櫥窗還漂亮。老闆,你不要笑我喔。」'
```

### 3. `e2000w_save` — 兩千萬拯救育幼院

原句「這輩子都會記得」太大、太工整。

建議改寫:

```ts
emotion: 'cry',
text: '「你、你是說真的?兩千萬不是兩千塊耶……我……我先打電話給院長,她一定以為我聽錯了。」'
```

### 4. `ending_protagonist` — 主角結局

原段落是戀愛結局模板,缺蘇嫚君的迷糊和 90s 小物。

建議改寫:

```ts
type: 'narration',
text: '蘇嫚君低頭看著手上的通告本,把夾在裡面的育幼院照片扶正。\n\n「那……明天開始,我還是會遲到一下下喔。可是我會先打電話。」\n\n她說完才像是想起什麼,紅著臉把手伸過來。\n\n「還有,以後不要只叫我蘇小姐了。」'
```

### 5. `e2000w_choice` — 危機選項

原選項像遊戲資源按鈕,不像人真的面對育幼院搬遷。

建議改寫:

```ts
options: [
  { label: '我去調這筆錢,先把育幼院保住。', next: 'e2000w_save', setFlags: { saved_orphanage: true, suman_friendship: 200 } },
  { label: '先找臨時安置,別讓孩子沒地方睡。', next: 'e2000w_lose' },
]
```

## 整體建議

1. 把 narration 從「情緒標籤」改成「可看見的動作/物件」:少用紅著眼眶、神色凝重、時間過很快,多用通知單、通告本、電話、報紙、外套皺褶。
2. 蘇嫚君的台詞多保留不完整句:用「啊」「嗯……」「不是啦」「我不是那個意思」增加天真迷糊感,避免每句都把事件資訊講完。
3. 每個育幼院事件至少放一個具體生活名詞:林媽媽、院長、義賣、鐵門、窗戶、卡帶、彩燈、公車、滷肉飯,讓 90s 台灣感落地。
4. Choice 避免純好壞二分:拒絕選項也可以有人味,例如工作排程、先安置、先確認狀況,不要只做冷酷分支。
5. 高情緒段落不要直接說「幸福」「一輩子記得」「一起走下去」,改用延遲反應、手忙腳亂、轉移話題、具體承諾來表現。
